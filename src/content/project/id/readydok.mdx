## Tentang proyek

**ReadyDok** adalah platform SaaS untuk klinik dan dokter di Indonesia. Pasien
bisa memesan janji secara online, klinik mengelola jadwal dan rekam medis, dan
praktik menumbuhkan retensi melalui pengingat otomatis serta profil publik
berisi ulasan.

Saya turut mendirikan ReadyDok di 2017 dan memimpin sisi engineering. Stack-nya
berkembang dari monolith Laravel menjadi backend Node.js + TypeScript dengan
frontend Vue/Nuxt, berjalan di platform cloud.

## Sorotan

- **Engine jadwal**: deteksi bentrok, slot berulang, aturan per-dokter.
- **Portal pasien**: alur booking mobile-first dengan pengingat lewat WhatsApp dan email.
- **Dashboard admin**: grid jadwal, riwayat pasien, pelaporan.
- **Integrasi**: pembayaran, pesan, dan sinkronisasi kalender.
